== Description ==
A simple calendar-based timecard solution. Record hours on weekly, bi-weekly, monthly or bi-monthly schedules, including automatic overtime calculations.  Dynamic Time is mobile compatible and integrates with existing WordPress users.

= Special Features =
* Automatic overtime calculations, configurable by user, even across pay periods
* Set the work week to begin on any day of the week
* Responsive layout for any screen size
* Automatic integration with existing WordPress users

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/dynamic-time` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Visit Dynamic Time from the WordPress admin menu to configure settings

== Frequently Asked Questions ==

= Does this plugin have a user limit? =
This plugin works with an unlimited number of users/employees.

= Can I lookup users by pay period? =
Currently all users are in one list, but upcoming features will allow filtering by pay period, user names, etc.
